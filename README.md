# AdoptMe Backend API

Este es un proyecto backend para gestión de adopción de mascotas.

---

## Tecnologías

- Node.js
- Express
- MongoDB
- Mongoose
- Mocha
- Supertest

---

## Cómo instalar

Abrir terminal y ejecutar:

npm install

---

## Cómo correr el proyecto

npm run dev

---

## Cómo correr tests

npm test

---

## Rutas principales

- /api/users
- /api/pets
- /api/adoptions
- /api/sessions

---

## Tests

Se testean endpoints de adopciones usando Supertest.
